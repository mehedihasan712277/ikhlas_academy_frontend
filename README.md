## To setup and run the project in your local machine

First, clone the repo
```bash
git clone git@github.com:mehedihasan712277/ikhlas_academy_frontend.git
```
then enter to the folder
```bash
cd ikhlas_academy_frontend
```
then instal all necessary packages
```bash
npm install
```
then run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
